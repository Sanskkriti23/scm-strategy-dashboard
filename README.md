# Streamlit SCM Strategy Dashboard (Shareable URL)

## Local run
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy (Streamlit Community Cloud)
1) Create a GitHub repo and upload these files:
- streamlit_app.py
- requirements.txt
- SCM_strategy_reformatted_validation_v2_GTM.docx

2) Go to https://share.streamlit.io
- Connect GitHub
- Select your repo
- Set **Main file path** to: streamlit_app.py
- Click Deploy

Streamlit will give you a URL anyone can open without running code.
